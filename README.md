# 📊 CSV Editor for PyQt

A modern desktop application built with **Python**, **PyQt5**, and **Pandas** for viewing, searching, editing, and managing CSV files with a clean and responsive interface.

> Designed for handling large CSV datasets such as drug databases, news archives, and categorized records.

---

## ✨ Features

- 📂 Browse and open multiple CSV files
- 🔍 Instant search and filtering
- ✏️ Edit CSV cells directly inside the table
- 💾 Save changes back to the original CSV
- 🗑️ Delete selected rows
- 📄 View complete row details
- 🌐 Open URLs directly in your browser
- 📋 Multi-tab interface
- 🌙 Modern Dark Theme
- ⚡ Fast performance using Pandas

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/alirezakohzadi/csv_editor_pyqt6.git
cd csv_editor_pyqt6
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
